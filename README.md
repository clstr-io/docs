# clstr.io

_Build distributed systems from scratch._

This repository contains the documentation, challenge definitions, and guides that power [clstr.io](https://clstr.io).

## Local Development

This site is built with [Astro](https://astro.build/) and [Starlight](https://starlight.astro.build/). To run locally:

```console
$ git clone https://github.com/clstr-io/docs.git
$ cd docs
$ pnpm install
$ pnpm dev
```

Visit [http://localhost:4321](http://localhost:4321) to preview.
