## How it works

<div class="lp-steps">
  <div class="lp-step">
    <span class="lp-step-num">01</span>
    <div>
      <strong>Pick a challenge</strong>
      <p>Each challenge is a real distributed system broken into stages. Start simple, add complexity one stage at a time.</p>
    </div>
  </div>
  <div class="lp-step">
    <span class="lp-step-num">02</span>
    <div>
      <strong>Build in any language</strong>
      <p>Go, Rust, Python, whatever. Update the <code>Dockerfile</code> to build and start your server. The tests only care about behavior.</p>
    </div>
  </div>
  <div class="lp-step">
    <span class="lp-step-num">03</span>
    <div>
      <strong>Run the tests</strong>
      <p>Run <code>clstr test</code>. Tests simulate network failures, crashes, and real-world conditions against your implementation.</p>
    </div>
  </div>
  <div class="lp-step">
    <span class="lp-step-num">04</span>
    <div>
      <strong>Advance</strong>
      <p>When you pass, run <code>clstr next</code>. Each stage unlocks after the previous one is solid.</p>
    </div>
  </div>
